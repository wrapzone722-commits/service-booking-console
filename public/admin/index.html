Кандидаты

Посты и расписание

Включение поста и сетка времени: зелёные слоты доступны для записи в приложении и виджете, серые — закрыты. Интервал задаёт шаг сетки (как в «Настройках», если не своё расписание).

Новости

Товары за баллы

Клиенты смогут обменивать баллы на эти товары/услуги в приложении.

Приглашения (QR)

Код привязан к услуге и посту: клиент вводит его в приложении при записи — цена будет 0 ₽. QR ведёт на страницу с подсказкой. Раздел защищён паролем администратора.

Клиенты

Типы автомобилей

Список для выбора в приложении (седан, кроссовер и т.д.). Нужен пароль администратора (как в «Настройках») для загрузки фото.

OpenClaw

Подключение персонального ассистента OpenClaw к этой консоли: тот же Admin API, что использует браузер (заголовок X-Admin-Key). Установка: npm i -g openclaw@latest → Getting started.

OpenClaw выполняется на вашей машине или сервере. Добавьте в workspace переменные SERVICE_BOOKING_ADMIN_URL и SERVICE_BOOKING_ADMIN_KEY (см. SKILL.md). Пароль не коммитьте.
