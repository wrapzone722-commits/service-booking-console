# Шрифты для актов выполненных работ (PDF)

Для корректного отображения кириллицы в PDF «Акт выполненных работ» поместите сюда TTF-шрифт с поддержкой русского языка.

**Рекомендуемый шрифт:** PT Sans (Google Fonts)  
- Скачайте [PT Sans](https://fonts.google.com/specimen/PT+Sans) и поместите файл `PTSans-Regular.ttf` в эту папку.

Если файл отсутствует, в акте будет использован стандартный шрифт Helvetica (кириллица может отображаться некорректно).
